Pig Latin!!

Welcome to the Pig Latin app! This project is a simple JavaScript application that translates English sentences into Pig Latin. Pig Latin is a fun and playful language game where words in English are altered according to a simple set of rules.

This app is hosted on AWS and specifically leverages S3 and Route 53.

How It Works

The rules for converting words to Pig Latin are as follows:

  • If a word begins with a consonant or consonant cluster, move it to the end of the word and add "ay". For example, "hello" becomes "ellohay".
  • If a word begins with a vowel, simply add "way" to the end of the word. For example, "apple" becomes "appleway".

Features

This project includes the following features:

  • Real-time translation of English sentences to Pig Latin.
  • Simple and intuitive user interface.

Getting Started

To get started, simply open the application and start typing your sentences in English, then click the translate button. The translation will be displayed instantly.
